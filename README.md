# Specimen tracker

A simple Streamlit app that allows for inventory and management of specimens for clearing and staining! 

[![Open in Streamlit]([https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://support-tickets-template.streamlit.app/](https://humble-invention-rvv6px994rj2p95v-8501.app.github.dev/))

### How to run it on your own machine

1. Install the requirements

   ```
   $ pip install -r requirements.txt
   ```

2. Run the app

   ```
   $ streamlit run streamlit_app.py
   ```
